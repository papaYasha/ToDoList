# ToDoList

App for notes

## Stack
* Xcode 12.5.1
* Minimum iOS Deployment Target: 13.0
* UIKit
* Chameleon (https://github.com/vicc/chameleon), *for multicolor cells*
* Realm
* SwipeCellKit(https://github.com/SwipeCellKit/SwipeCellKit), *right swipe delete cell*
***
